# 11-Negotiation

## Salary Negotiation Framework POC

This POC provides a comprehensive framework for evaluating job offers and negotiating compensation packages targeting ₹70L+ for AI/Data Architect roles. It includes market research tools, offer comparison spreadsheets, and negotiation strategy guides.

### Key Features
- Offer evaluation and comparison framework
- Market salary research automation
- Negotiation strategy simulator
- Total compensation analysis (CTC vs in-hand)
- Counter-offer preparation tools

### Tech Stack
- Python 3.8+
- Streamlit for interactive negotiation simulator
- Pandas for data analysis and comparison
- Plotly for compensation visualization
- SQLite for offer database
- Web scraping for market data
- Excel/PDF export capabilities

### Quick Start
```bash
# Setup environment
git clone <your-repo-url>
cd 11-Negotiation
pip install -r requirements.txt

# Run negotiation framework
streamlit run src/app.py

# Research market rates
python src/market_research.py --role "AI Architect" --location "Bangalore"
```

### Framework Components
- **Offer Analyzer**: Break down and compare offer components
- **Market Research**: Automated salary data collection
- **Negotiation Simulator**: Practice negotiation scenarios
- **Strategy Builder**: Personalized negotiation approaches
- **Progress Tracker**: Negotiation outcome analytics
