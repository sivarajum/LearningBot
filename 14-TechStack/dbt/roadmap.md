# dbt - Learning Roadmap

## 🗺️ Complete Learning Path from Beginner to Expert

Master dbt (data build tool) for analytics engineering.

---

## 📅 6-Week Intensive Roadmap

### Week 1-2: Foundations 🟢
- [ ] **Day 1-2: Installation & Setup** - dbt installation, project setup
- [ ] **Day 3-4: Core Concepts** - Models, sources, tests
- [ ] **Day 5-7: Basic Models** - SELECT statements, ref() function
- [ ] **Day 8-10: Jinja Templating** - Variables, macros, loops
- [ ] **Day 11-12: Sources** - Source definitions, freshness tests
- [ ] **Day 13-14: Tests** - Data tests, schema tests

**Milestone**: Can build basic dbt models

---

### Week 3-4: Production Patterns 🟡
- [ ] **Day 15-16: Advanced Models** - Incremental models, snapshots
- [ ] **Day 17-18: Macros** - Custom macros, package macros
- [ ] **Day 19-21: Documentation** - Model documentation, docs generation
- [ ] **Day 22-23: Seeds** - Seed data, CSV files
- [ ] **Day 24-25: Hooks** - Pre-hooks, post-hooks
- [ ] **Day 26-28: Best Practices** - Project organization, naming

**Milestone**: Production-ready dbt projects

---

### Week 5-6: Advanced Features 🔴
- [ ] **Day 29-30: dbt Cloud** - Cloud IDE, scheduling, CI/CD
- [ ] **Day 31-32: Advanced Jinja** - Complex macros, packages
- [ ] **Day 33-35: Testing Strategy** - Comprehensive testing
- [ ] **Day 36-37: Performance** - Optimization, materialization strategies
- [ ] **Day 38-39: Integration** - Airflow, orchestration, monitoring
- [ ] **Day 40-42: Capstone Project** - Enterprise analytics project

**Milestone**: dbt expert

---

## 🎯 Skill Levels
- **🟢 Beginner**: Basic models, tests, Jinja
- **🟡 Intermediate**: Advanced models, macros, documentation
- **🔴 Advanced**: dbt Cloud, optimization, integration
