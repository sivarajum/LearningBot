# Vertex AI Roadmap – 6 Weeks

## Weeks 1-2: Foundations (🟢)
- [ ] Day 1: Enable API; roles/IAM basics
- [ ] Day 2: Endpoints/models overview; prebuilt predictions
- [ ] Day 3: Datasets and training jobs basics
- [ ] Day 4: Batch vs online prediction
- [ ] Day 5: Model Registry basics
- [ ] Day 6: Notebook/Workbench quickstart
- [ ] Day 7: Mini-project: deploy prebuilt model and predict
- [ ] Day 8: Logging/monitoring basics
- [ ] Day 9: CMEK and network options overview
- [ ] Day 10: Review + tidy IAM
- [ ] Day 11-12: Feature Store intro
- [ ] Day 13-14: Cost basics (batch vs online)

**Milestone**: Run and deploy a prebuilt model with monitoring basics.

## Weeks 3-4: Intermediate (🟡)
- [ ] Day 15: Custom training jobs; custom containers
- [ ] Day 16: Pipelines basics (TFX/Kubeflow/custom components)
- [ ] Day 17: CI/CD flow: build -> train -> eval -> register -> deploy
- [ ] Day 18: Traffic splitting; canary deploys
- [ ] Day 19: Model monitoring: skew/drift alerts
- [ ] Day 20: Batch predictions best practices
- [ ] Day 21: Feature Store online/offline sync; TTL/backfills
- [ ] Day 22-23: Mini-project: pipeline to train+deploy
- [ ] Day 24-25: Security: IAM per resource, CMEK, VPC-SC
- [ ] Day 26-28: Cost: right-size accelerators, schedule jobs

**Milestone**: Custom training+deployment with monitoring and security.

## Weeks 5-6: Advanced (🔴)
- [ ] Day 29: Advanced pipelines (caching, conditional steps)
- [ ] Day 30: Multi-model endpoints; autoscaling tuning
- [ ] Day 31: Governance: lineage, audit logs, versioning policies
- [ ] Day 32: DR/rollback strategies; safe deploy patterns
- [ ] Day 33: MLOps maturity: tests, metrics, quality gates
- [ ] Day 34: Serve-time optimization: request batching, latency SLOs
- [ ] Day 35: Data privacy: PII handling, encryption, access reviews
- [ ] Day 36: Observability deep dive: traces/metrics/logs
- [ ] Day 37-38: Capstone: full MLOps pipeline with canary and monitoring
- [ ] Day 39-42: Documentation, runbooks, cost/perf reports

**Milestone**: Governed, monitored, cost-aware Vertex AI platform.
