# GCP Interactive Visualizations

This directory contains interactive, animated visualizations for Google Cloud Platform services that complement the static documentation.

## Files

### `interactive-architecture.html`
- **Purpose**: General GCP architecture overview with floating service components
- **Features**:
  - Animated floating GCP service icons
  - Particle background effects
  - Interactive service selection with information panels
  - Real-time scaling indicators
  - Connection lines showing service relationships
  - Responsive design for mobile devices

### `data-flow-animation.html`
- **Purpose**: Detailed data flow visualization showing how data moves between GCP services
- **Features**:
  - Animated data packets flowing between services
  - Real-time metrics display for each service
  - Interactive workflow steps
  - Status indicators and connection animations
  - Progress tracking for workflow completion
  - Speed controls (pause/resume, speed up/slow down)
  - Keyboard shortcuts for navigation

## How to Use

1. **Open in Browser**: Simply open any `.html` file in a modern web browser
2. **No Dependencies**: These are self-contained HTML files with embedded CSS and JavaScript
3. **Interactive Features**:
   - Click on service nodes to see details
   - Use control buttons to pause/resume animations
   - Press keyboard shortcuts for quick actions
   - View real-time metrics and status updates

## Keyboard Shortcuts (Data Flow Animation)

- **Space**: Pause/Resume data flow
- **W**: Show workflow steps
- **R**: Reset demonstration
- **+ / =**: Speed up animations
- **-**: Slow down animations
