Hyperparameter Tuning - Automated Optimization

Advanced

Hyperparameter tuning automatically finds the best model configuration by testing different combinations of hyperparameters. Vertex AI uses Bayesian optimization and parallel execution for efficient tuning.

🎛️ Tuning Strategies
  • Bayesian Optimization: Intelligent search using probabilistic models
  • Grid Search: Exhaustive search over specified parameter values
  • Random Search: Random sampling from parameter distributions
  • Parallel Execution: Run multiple trials simultaneously
  • Early Stopping: Stop unpromising trials early
1from google.cloud import aiplatform
2from google.cloud.aiplatform import hyperparameter_tuning as hpt
3
4def create_hyperparameter_tuning_job(project_id, location, training_script):
5 """Create a hyperparameter tuning job."""
6 aiplatform.init(project=project_id, location=location)
7
8 # Define hyperparameter tuning job
9 tuning_job = aiplatform.HyperparameterTuningJob(
10 display_name="pytorch-hyperparameter-tuning",
11 custom_job=aiplatform.CustomTrainingJob(
12 display_name="pytorch-training",
13 script_path=training_script,
14 container_uri="gcr.io/cloud-aiplatform/training/pytorch-gpu.1-12:latest",
15 requirements=["torch==1.12.0", "torchvision"]
16 ),
17 metric_spec={
18 "accuracy": "maximize"
19 },
20 parameter_spec={
21 "learning-rate": hpt.DoubleParameterSpec(min=0.001, max=0.1, scale="log"),
22 "batch-size": hpt.DiscreteParameterSpec(values=[16, 32, 64], scale="linear"),
23 "num-layers": hpt.IntegerParameterSpec(min=2, max=5, scale="linear")
24 },
25 max_trial_count=20,
26 parallel_trial_count=5,
27 max_failed_trial_count=3
28 )
29
30 # Run tuning job
31 tuning_job.run()
32
33 # Get best trial
34 best_trial = tuning_job.trials[0]
35 for trial in tuning_job.trials[1:]:
36 if trial.final_measurement.metrics[0].value > best_trial.final_measurement.metrics[0].value:
37 best_trial = trial
38
39 print(f"Best trial parameters: {best_trial.parameters}")
40 print(f"Best accuracy: {best_trial.final_measurement.metrics[0].value}")
41
42 return tuning_job

RAG Applications - Retrieval-Augmented Generation

Advanced

Retrieval-Augmented Generation (RAG) combines the power of large language models with information retrieval to provide accurate, up-to-date responses based on your documents.

📚 RAG Architecture
  • Document Ingestion: Process and chunk documents for retrieval
  • Vector Embeddings: Convert text to semantic vectors
  • Similarity Search: Find relevant documents for queries
  • Context Generation: Use retrieved docs to enhance LLM responses
  • Source Attribution: Cite sources for generated answers
1from vertexai.language_models import TextEmbeddingModel, TextGenerationModel
2from google.cloud import aiplatform
3from sklearn.metrics.pairwise import cosine_similarity
4import numpy as np
5
6class RAGSystem:
7 """Simple RAG system using Vertex AI."""
8
9 def __init__(self, project_id, location):
10 vertexai.init(project=project_id, location=location)
11 self.embedding_model = TextEmbeddingModel.from_pretrained("textembedding-gecko@003")
12 self.generation_model = TextGenerationModel.from_pretrained("text-bison@001")
13 self.documents = []
14 self.embeddings = []
15
16 def add_documents(self, documents):
17 """Add documents to the knowledge base."""
18 self.documents.extend(documents)
19 # Generate embeddings for new documents
20 new_embeddings = self.embedding_model.get_embeddings(documents)
21 new_embedding_vectors = [embedding.values for embedding in new_embeddings]
22 self.embeddings.extend(new_embedding_vectors)
23
24 def retrieve_relevant_docs(self, query, top_k=3):
25 """Retrieve most relevant documents for a query."""
26 # Generate embedding for query
27 query_embedding = self.embedding_model.get_embeddings([query])[0].values
28
29 # Calculate similarities
30 similarities = cosine_similarity([query_embedding], self.embeddings)[0]
31
32 # Get top-k most similar documents
33 top_indices = np.argsort(similarities)[::-1][:top_k]
34 relevant_docs = [self.documents[i] for i in top_indices]
35
36 return relevant_docs
37
38 def generate_answer(self, query):
39 """Generate an answer using RAG."""
40 # Retrieve relevant documents
41 relevant_docs = self.retrieve_relevant_docs(query)
42
43 # Create context from retrieved documents
44 context = "\n\n".join(relevant_docs)
45
46 # Create prompt with context
47 prompt = f"""
48Based on the following context, answer the question accurately.
49If you cannot find the answer in the context, say so.
50
51Context:
52{context}
53
54Question: {query}
55
56Answer:
57"""
58
59 # Generate answer
60 response = self.generation_model.predict(
61 prompt=prompt,
62 max_output_tokens=256,
63 temperature=0.1
64 )
65
66 return response.text
