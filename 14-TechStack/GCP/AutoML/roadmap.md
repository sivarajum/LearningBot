# AutoML Roadmap – 6 Weeks

## Weeks 1-2: Foundations (🟢)
- [ ] Day 1: Enable API; explore products (Vision/NLP/Tables/Translation/Video)
- [ ] Day 2: Data requirements per product
- [ ] Day 3: Import dataset (GCS/BQ) for one use case
- [ ] Day 4: Train first model; review metrics
- [ ] Day 5: Tune labels/data; retrain
- [ ] Day 6: Batch vs online prediction basics
- [ ] Day 7: Mini-project: end-to-end Vision or NLP model
- [ ] Day 8: IAM basics; roles per product
- [ ] Day 9: Export options; storage locations
- [ ] Day 10: Review + cleanup datasets
- [ ] Day 11-12: Costs: training vs prediction
- [ ] Day 13-14: Evaluation best practices; per-class metrics

**Milestone**: Working AutoML model with evaluated metrics.

## Weeks 3-4: Intermediate (🟡)
- [ ] Day 15: Balanced datasets; label quality; leakage checks
- [ ] Day 16: Threshold tuning; precision/recall tradeoffs
- [ ] Day 17: Batch prediction pipelines (to BQ/Storage)
- [ ] Day 18: Online endpoints; traffic split/rollback
- [ ] Day 19: Monitoring predictions; drift signals
- [ ] Day 20: Governance: model cards, dataset versions
- [ ] Day 21: Security: IAM least privilege; CMEK support review
- [ ] Day 22-23: Mini-project: batch + online deploy with rollback
- [ ] Day 24-25: Cost controls: schedule batch, cleanup artifacts
- [ ] Day 26-28: Integrate with pipelines for retrain

**Milestone**: Deployable AutoML with monitoring and governance basics.

## Weeks 5-6: Advanced (🔴)
- [ ] Day 29: Multi-product strategy (vision+text+tables)
- [ ] Day 30: VPC-SC and data residency checks
- [ ] Day 31: Edge/vertex export (where supported)
- [ ] Day 32: Human-in-loop for low-confidence cases
- [ ] Day 33: A/B tests; champion/challenger
- [ ] Day 34: Compliance: audit logs, data retention
- [ ] Day 35: DR: backup datasets/models; reproducibility
- [ ] Day 36: Performance: latency/throughput tests on endpoints
- [ ] Day 37-38: Capstone: governed AutoML with retrain pipeline
- [ ] Day 39-42: Documentation, SOPs, cost/perf reports

**Milestone**: Governed, monitored, multi-use AutoML with retrain pipeline.
