## Beginner Level Questions

### What is machine learning and how does it differ from traditional programming?

**Answer:**

Machine learning is a method where algorithms learn patterns from data to make predictions or decisions, rather than following explicit instructions. Traditional programming: input + rules → output. ML: input + output → rules (model). ML adapts and improves with more data.

## Intermediate Level Questions

### Explain the difference between supervised, unsupervised, and reinforcement learning.

**Answer:**

Supervised: Learn from labeled data (input-output pairs). Examples: classification, regression. Unsupervised: Find patterns in unlabeled data. Examples: clustering, dimensionality reduction. Reinforcement: Learn through interaction with environment, receiving rewards/penalties. Examples: game playing, robotics.
