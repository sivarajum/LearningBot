## Beginner Level Questions

### What is SQLite and when should you use it?

**Answer:**

SQLite is a file-based, serverless SQL database. Use it for: development/testing, embedded applications, mobile apps, small to medium applications, read-heavy workloads, and when you don't need concurrent writes. Avoid for: high-concurrency write scenarios, large datasets requiring distribution, or when you need advanced features like stored procedures.
