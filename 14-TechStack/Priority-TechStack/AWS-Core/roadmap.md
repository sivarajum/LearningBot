# AWS Core Services - Learning Roadmap

## 🗺️ Complete Learning Path from Beginner to Expert

Master AWS Core Services: S3, EC2, Lambda, IAM, and more for cloud data engineering.

---

## 📅 8-Week Intensive Roadmap

### Week 1-2: Foundations 🟢
- [ ] **Day 1-2: AWS Account Setup** - Account creation, IAM basics
- [ ] **Day 3-4: S3 (Storage)** - Buckets, objects, versioning, lifecycle
- [ ] **Day 5-7: EC2 (Compute)** - Instances, AMIs, security groups
- [ ] **Day 8-10: IAM (Security)** - Users, roles, policies, best practices
- [ ] **Day 11-12: VPC (Networking)** - Subnets, route tables, security
- [ ] **Day 13-14: CloudWatch (Monitoring)** - Logs, metrics, alarms

**Milestone**: Can use core AWS services

---

### Week 3-4: Data Services 🟡
- [ ] **Day 15-16: Lambda (Serverless)** - Functions, triggers, integrations
- [ ] **Day 17-18: Glue (ETL)** - Crawlers, jobs, Data Catalog
- [ ] **Day 19-21: EMR (Big Data)** - Spark clusters, Hadoop
- [ ] **Day 22-23: RDS (Databases)** - Managed databases, backups
- [ ] **Day 24-25: Redshift (Data Warehouse)** - Clusters, queries, optimization
- [ ] **Day 26-28: Kinesis (Streaming)** - Streams, Firehose, Analytics

**Milestone**: Can build data pipelines on AWS

---

### Week 5-6: Advanced Services 🔴
- [ ] **Day 29-30: Step Functions** - Workflow orchestration
- [ ] **Day 31-32: EventBridge** - Event-driven architecture
- [ ] **Day 33-35: Sagemaker** - ML platform, training, deployment
- [ ] **Day 36-37: Cost Optimization** - Cost Explorer, Reserved Instances
- [ ] **Day 38-39: Security Advanced** - KMS, Secrets Manager, compliance
- [ ] **Day 40-42: Architecture** - Well-Architected Framework

**Milestone**: Advanced AWS practitioner

---

### Week 7-8: Mastery 🏆
- [ ] **Day 43-44: Multi-Account** - Organizations, Control Tower
- [ ] **Day 45-46: DevOps** - CodePipeline, CodeBuild, Infrastructure as Code
- [ ] **Day 47-49: Advanced Patterns** - Serverless, event-driven, microservices
- [ ] **Day 50-51: Troubleshooting** - Debugging, monitoring, optimization
- [ ] **Day 52-53: Certification Prep** - AWS Certified Solutions Architect
- [ ] **Day 54-56: Capstone Project** - Enterprise cloud platform

**Milestone**: AWS expert

---

## 🎯 Skill Levels
- **🟢 Beginner**: Core services (S3, EC2, IAM)
- **🟡 Intermediate**: Data services (Glue, EMR, Redshift)
- **🔴 Advanced**: Advanced services (Sagemaker, Step Functions)
- **🏆 Expert**: Architecture, multi-account, enterprise
