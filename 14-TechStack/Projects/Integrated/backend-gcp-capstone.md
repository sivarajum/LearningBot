# Capstone: E-commerce API on Cloud Run
- Services on Cloud Run; CI/CD via Cloud Build; artifact in AR.
- Data: Cloud SQL (private IP/IAM) or Spanner; migrations.
- Networking: HTTPS LB, SSL, custom domain; WAF rules.
- Reliability: canary/blue-green; rollback drill; load test results.
- Observability: structured logs + traces; dashboards; burn-rate alerts.
- Security: secrets in SM; least privilege; restricted ingress/egress.
