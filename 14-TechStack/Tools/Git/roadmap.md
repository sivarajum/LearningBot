# Git - Learning Roadmap

## 🗺️ Complete Learning Path from Beginner to Expert

Master Git, the distributed version control system.

---

## 📅 4-Week Intensive Roadmap

### Week 1: Foundations 🟢
- [ ] **Day 1-2: Installation & Setup** - Git installation, configuration
- [ ] **Day 3-4: Basic Commands** - init, add, commit, status, log
- [ ] **Day 5-7: Branching** - Create, switch, merge branches
- [ ] **Day 8-10: Remote Repositories** - clone, push, pull, remote
- [ ] **Day 11-12: Collaboration** - Fork, pull requests, code review
- [ ] **Day 13-14: Undoing Changes** - reset, revert, checkout

**Milestone**: Can use Git for version control

---

### Week 2: Intermediate 🟡
- [ ] **Day 15-16: Advanced Branching** - Rebase, cherry-pick, merge strategies
- [ ] **Day 17-18: Stashing** - Stash, pop, apply, stash management
- [ ] **Day 19-21: Tags** - Creating tags, annotated tags, releases
- [ ] **Day 22-23: Hooks** - Pre-commit, post-commit, custom hooks
- [ ] **Day 24-25: Submodules** - Git submodules, managing dependencies
- [ ] **Day 26-28: Workflows** - Git flow, GitHub flow, GitLab flow

**Milestone**: Intermediate Git user

---

### Week 3-4: Advanced Features 🔴
- [ ] **Day 29-30: Advanced Commands** - Reflog, bisect, filter-branch
- [ ] **Day 31-32: Git Internals** - Objects, refs, index, HEAD
- [ ] **Day 33-35: Performance** - Large repos, shallow clones, sparse checkout
- [ ] **Day 36-37: Troubleshooting** - Common issues, recovery, debugging
- [ ] **Day 38-39: Best Practices** - Commit messages, branching strategy
- [ ] **Day 40-42: CI/CD Integration** - GitHub Actions, GitLab CI, automation

**Milestone**: Git expert

---

## 🎯 Skill Levels
- **🟢 Beginner**: Basic commands, branching, collaboration
- **🟡 Intermediate**: Advanced branching, workflows, hooks
- **🔴 Advanced**: Internals, performance, troubleshooting
