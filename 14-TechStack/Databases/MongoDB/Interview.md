## Beginner Level Questions

### What is MongoDB and how does it differ from relational databases?

**Answer:**

MongoDB is a NoSQL document database. Differences: stores documents (BSON) vs tables/rows, flexible schema vs fixed schema, horizontal scaling vs vertical scaling, embedded documents vs normalized tables, JavaScript query language vs SQL. Use MongoDB for flexible schemas, rapid development, and horizontal scaling needs.

## Intermediate Level Questions

### Explain MongoDB indexing and how to optimize queries.

**Answer:**

MongoDB uses B-tree indexes. Create indexes on frequently queried fields, compound indexes for multi-field queries, text indexes for full-text search. Use explain() to analyze query performance. Indexes improve read performance but slow writes. Monitor index usage and remove unused indexes.
