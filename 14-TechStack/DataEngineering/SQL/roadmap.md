# SQL for Data Engineering - Learning Roadmap

## 🗺️ Complete Learning Path from Beginner to Expert

Master SQL for data engineering and analytics.

---

## 📅 6-Week Intensive Roadmap

### Week 1-2: Foundations 🟢
- [ ] **Day 1-2: SQL Basics** - SELECT, FROM, WHERE, ORDER BY
- [ ] **Day 3-4: Joins** - INNER, LEFT, RIGHT, FULL OUTER joins
- [ ] **Day 5-7: Aggregations** - GROUP BY, HAVING, aggregate functions
- [ ] **Day 8-10: Subqueries** - Correlated, non-correlated subqueries
- [ ] **Day 11-12: Window Functions** - ROW_NUMBER, RANK, LAG, LEAD
- [ ] **Day 13-14: CTEs** - Common Table Expressions, recursive CTEs

**Milestone**: Can write complex SQL queries

---

### Week 3-4: Production Patterns 🟡
- [ ] **Day 15-16: DDL** - CREATE, ALTER, DROP tables, indexes
- [ ] **Day 17-18: DML** - INSERT, UPDATE, DELETE, MERGE
- [ ] **Day 19-21: Performance** - Query optimization, indexing strategies
- [ ] **Day 22-23: Advanced Functions** - String, date, numeric functions
- [ ] **Day 24-25: Transactions** - ACID, isolation levels, locking
- [ ] **Day 26-28: Data Types** - JSON, arrays, custom types

**Milestone**: Production-ready SQL skills

---

### Week 5-6: Advanced Features 🔴
- [ ] **Day 29-30: Advanced Queries** - Pivoting, unpivoting, hierarchical queries
- [ ] **Day 31-32: Stored Procedures** - Functions, procedures, triggers
- [ ] **Day 33-35: Data Warehousing** - Star schema, fact tables, dimensions
- [ ] **Day 36-37: ETL Patterns** - Incremental loads, change data capture
- [ ] **Day 38-39: Best Practices** - Code organization, optimization
- [ ] **Day 40-42: Capstone Project** - Complex data engineering queries

**Milestone**: SQL data engineering expert

---

## 🎯 Skill Levels
- **🟢 Beginner**: Basic SELECT, JOINs, aggregations
- **🟡 Intermediate**: Window functions, optimization, DDL/DML
- **🔴 Advanced**: Stored procedures, data warehousing, ETL
