# React Roadmap – 6 Weeks

## Weeks 1-2: Foundations (🟢)
- [ ] Day 1: JSX, components, props/state
- [ ] Day 2: Event handling; conditional rendering; lists/keys
- [ ] Day 3: Hooks basics (useState/useEffect)
- [ ] Day 4: Forms + controlled inputs
- [ ] Day 5: Styling options (CSS modules/CSS-in-JS)
- [ ] Day 6: Project setup with CRA/Vite/Next
- [ ] Day 7: Mini-project: simple dashboard

- [ ] Day 19: Error boundaries; fallbacks
- [ ] Day 20: Performance profiling; virtualized lists
- [ ] Day 21: Design system/theming; component library integration
- [ ] Day 22: State modeling; prop drilling vs context
- [ ] Day 23: Accessibility basics (a11y lint, focus states)

- [ ] Day 26-28: Harden tests; add e2e (Playwright/Cypress)

**Milestone**: Production-ready patterns with caching, forms, testing.

## Weeks 5-6: Advanced (🔴)
- [ ] Day 29: Advanced Suspense patterns; transitions
- [ ] Day 30: Error monitoring + web vitals
- [ ] Day 31: Runtime config; env handling; feature flags
- [ ] Day 32: Security: avoid secret leakage; sanitization
- [ ] Day 33: Performance budgets; bundle analysis
- [ ] Day 34: SSR/SSG tradeoffs (Next.js); edge rendering basics
- [ ] Day 35: Architecture: feature folders, module boundaries
- [ ] Day 36: API layer patterns; retry/backoff; auth flows
- [ ] Day 37-38: Capstone: full app with CI (lint/test/build), a11y checks
- [ ] Day 39-42: Documentation, handover, postmortems

**Milestone**: Deployable, observable, performant React app with CI/CD.
