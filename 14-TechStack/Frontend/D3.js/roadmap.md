# D3.js Roadmap – 6 Weeks

## Weeks 1-2: Foundations (🟢)
- [ ] Day 1: Selections; enter/update/exit
- [ ] Day 2: Scales (linear/band/time); axes
- [ ] Day 3: Basic shapes (rect/circle/path); SVG coords
- [ ] Day 4: Bar chart; margins and labels
- [ ] Day 5: Line chart; timeParse/timeFormat
- [ ] Day 6: Color scales; legends
- [ ] Day 7: Transitions basics
- [ ] Day 8: Data loading (csv/json); parsing
- [ ] Day 9: Tooltips and hover states
- [ ] Day 10: Responsive viewBox
- [ ] Day 11-12: Mini-project: combined bar/line
- [ ] Day 13-14: Review and refactor

**Milestone**: Build basic responsive charts with tooltips.

## Weeks 3-4: Intermediate (🟡)
- [ ] Day 15: Zoom/pan (d3-zoom); brushing
- [ ] Day 16: Scales/axes helpers; formatting
- [ ] Day 17: Layouts (stacked/area/pie)
- [ ] Day 18: Performance: reduce DOM nodes
- [ ] Day 19: Reusable chart functions; config props
- [ ] Day 20: Data transforms and testing
- [ ] Day 21: Accessibility: titles/desc/roles
- [ ] Day 22: Interactions: click/hover/keyboard focus
- [ ] Day 23-24: Mini-project: interactive dashboard
- [ ] Day 25-27: Integrate with React/Vue; wrapper component
- [ ] Day 28: Theming via CSS vars

**Milestone**: Interactive dashboard with reusable charts.

## Weeks 5-6: Advanced (🔴)
- [ ] Day 29: Canvas rendering for large data
- [ ] Day 30: Hybrid SVG+Canvas patterns
- [ ] Day 31: Animations performance; throttling
- [ ] Day 32: Complex layouts (treemap/force/sankey)
- [ ] Day 33: Export/printing/static render
- [ ] Day 34: Observability: capture errors, perf metrics
- [ ] Day 35: Security: sanitize data-driven attrs/text
- [ ] Day 36: Governance: lint rules, testing transforms
- [ ] Day 37-38: Capstone: multi-view, zoomable, accessible viz
- [ ] Day 39-42: Docs, runbooks, handoff

**Milestone**: Production-grade, performant, accessible visualizations.
