# 01: ML Fundamentals - Mini-Projects

## Overview
This repository contains three foundational machine learning mini-projects demonstrating core supervised learning concepts. Perfect for building portfolio credibility when transitioning from data engineering to AI/ML roles.

## Projects Included
1. **Iris Classification** - Multi-class classification using scikit-learn
2. **Movie Recommendation System** - Collaborative filtering implementation
3. **Sentiment Analysis** - Text classification for social media data

## Tech Stack
- Python 3.8+
- scikit-learn, pandas, numpy
- matplotlib, seaborn
- Jupyter Notebook

## Key Features
- Clean, documented code with proper READMEs
- Model evaluation with >85% accuracy targets
- Visualizations for model performance analysis
- GitHub-ready portfolio projects

## Getting Started
```bash
pip install -r requirements.txt
jupyter notebook
```

## Success Criteria
- All models achieve >85% test accuracy
- Professional code documentation
- Visual performance reports
- Ready for GitHub portfolio
